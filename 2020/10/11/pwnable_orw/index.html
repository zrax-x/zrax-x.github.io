pwnable.tw orw

  • PWN
  • shellcode

程序开了沙箱，只能使用open，read，write函数。因此整个流程就是先打开文件，然后读取内容，最后把内容打印出来。

这里需要了解每一个系统调用的参数设置。

open

eax = 0x05 系统调用号

ebx = filename 文件名

ecx = flags 置零即可

edx = mode 置零即可

返回值eax = fd

read

eax = 0x03 系统调用号

ebx = fd 文件指针

ecx = buf 缓冲区，存访读取内容

edx = len 读取的字节数

write

eax = 0x04 系统调用号

ebx = fd 文件指针，标准输出对应1

ecx = buf

edx = len
